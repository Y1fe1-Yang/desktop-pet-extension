# 桌面宠物 Chrome 扩展 - 安装指南

## 🚀 快速安装（3步）

### 步骤 1: 下载扩展

点击GitHub页面上的绿色 **"Code"** 按钮，然后选择 **"Download ZIP"**

或者直接下载：
```
https://github.com/YOUR_USERNAME/desktop-pet-extension/archive/refs/heads/main.zip
```

### 步骤 2: 解压文件

解压下载的ZIP文件到任意位置（比如桌面）

你会得到一个 `desktop-pet-extension-main` 文件夹

### 步骤 3: 安装到Chrome

1. 打开 Chrome 浏览器
2. 地址栏输入: `chrome://extensions/`
3. 打开右上角的 **"开发者模式"** 开关
4. 点击 **"加载已解压的扩展程序"**
5. 选择解压后的文件夹 `desktop-pet-extension-main`
6. 完成！

---

## ✅ 验证安装

安装成功后：
- 扩展列表中会出现 "Desktop Pet"
- 浏览器工具栏有扩展图标
- 打开任意网页，页面右下角会出现**彩色方块**（测试宠物）

### 基础交互
- **拖拽** - 按住鼠标移动宠物

## 🎨 如何添加真实宠物图片

要看到真正的卡皮巴拉或其他动物：

1. 准备宠物图片（PNG格式，透明背景）
2. 生成精灵图（sprite sheets）
3. 将 `sprite_*.png` 文件放到扩展文件夹
4. 在 `chrome://extensions/` 页面点击扩展的 **"重新加载"** 按钮
5. 刷新网页，看到真实宠物

## ❓ 常见问题

### Q1: "加载已解压的扩展程序"按钮是灰色的？
**A**: 请先打开右上角的 **"开发者模式"** 开关！

### Q2: 安装成功但网页上没有宠物？
**A**:
1. 刷新网页（F5）
2. 检查扩展是否启用（图标应该有颜色）
3. 打开控制台（F12）查看是否有错误

### Q3: 如何卸载？
**A**:
1. 打开 `chrome://extensions/`
2. 找到 "Desktop Pet" 扩展
3. 点击 **"移除"** 按钮

### Q4: 宠物会影响网页正常使用吗？
**A**: 不会。宠物：
- 有独立的层级，不会遮挡页面元素
- 不会影响页面功能
- 可以随时拖到角落

### Q5: 能在所有网站使用吗？
**A**: 可以在大部分网站使用，少数安全限制严格的网站（如 `chrome://` 页面）无法使用。
